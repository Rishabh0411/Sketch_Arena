# Quick Start Guide

## Start the Application

### Terminal 1 - Backend
```bash
cd backend
npm run dev
```

### Terminal 2 - Frontend
```bash
cd frontend  
npm run dev
```

## Play the Game

1. Open http://localhost:5173
2. Enter your name and create/join a room
3. Host clicks "Start Game"
4. Drawer selects a word
5. Drawer uses tools to draw (colors, pen, eraser, sizes)
6. Other players type guesses in chat
7. Correct guesses award points!

## What's Fixed

### Drawing Tools (NEW!)
- Pen tool with 8 colors
- Eraser tool

### Chat (FIXED!)
- Messages appear in styled boxes
